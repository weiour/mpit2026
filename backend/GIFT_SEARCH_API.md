# API для поиска подарков

## Обзор

Сервис поиска подарков позволяет находить реальные товары в популярных маркетплейсах с **корректными ценами**, скидками и предоставляет рабочие ссылки на них.

## Доступные источники

1. **Wildberries** - прямой поиск через API с точными ценами и скидками
2. **Ozon** - улучшенный веб-скрапинг с извлечением реальных цен
3. **Google Shopping** - через Custom Search API (требует API ключ)
4. **Яндекс Маркет** - запасной вариант с прямыми ссылками

## Особенности работы с ценами

### Wildberries
- **Точные цены** из официального API (включая копейки)
- **Скидки**: извлекаются `salePriceU` (текущая) и `priceU` (оригинальная)
- **Валидация**: цены < 10₽ отфильтровываются как некорректные
- **Конвертация**: автоматическое преобразование из копеек в рубли

### Ozon
- **Многослойный парсинг**: window.__STATE__ → LD+JSON → Fallback
- **Разные форматы цен**: `priceU`, `finalPriceU`, `salePriceU`
- **Скидки**: вычисляются автоматически при наличии `originalPrice`
- **Валидация**: проверка корректности URL изображений

## Endpoints

### 1. Поиск подарков

#### POST /gifts/search
```json
{
  "query": "беспроводные наушники",
  "budget": 5000,
  "limit": 10,
  "include_discounts": true,
  "min_discount": 10
}
```

#### GET /gifts/search
```
?query=беспроводные наушники&budget=5000&limit=10
```

#### Расширенная фильтрация (POST метод)
```json
{
  "query": "наушники",
  "budget": 5000,
  "limit": 10,
  "filter": {
    "min_price": 1000,
    "max_price": 5000,
    "stores": ["Wildberries", "Ozon"],
    "categories": ["Электроника"],
    "min_rating": 4.0,
    "has_discount": true,
    "min_discount_percent": 15
  }
}
```

### 2. Рекомендации для события

#### GET /events/{event_id}/gift-recommendations
```
/events/123/gift-recommendations?budget=10000&limit=15
```

### 3. Альтернативы для элемента вишлиста

#### GET /wishlist/{item_id}/alternatives
```
/wishlist/456/alternatives?budget=3000&limit=5
```

### 4. Дополнительные эндпоинты

#### GET /stores
Возвращает список доступных магазинов: `["Wildberries", "Ozon", "Яндекс Маркет", "Google Shopping"]`

#### GET /categories
Возвращает популярные категории подарков.

## Валидация цен

Система автоматически проверяет:

- **Минимальная цена**: 10₽ (отфильтровывает некорректные данные)
- **Максимальная цена**: 10,000,000₽ (отфильтровывает ошибки)
- **Тип данных**: цена должна быть числом
- **Скидки**: процент должен быть в диапазоне 0-100%
- **Бюджет**: цены выше бюджета отфильтровываются

## Обработка ошибок

- **Graceful degradation**: если один источник недоступен, используются другие
- **Fallback результаты**: прямые ссылки на страницы поиска
- **Валидация**: некорректные данные отфильтровываются
- **Логирование**: ошибки выводятся в консоль для отладки

Добавьте в `.env` файл:
```
GOOGLE_API_KEY=your_google_api_key
GOOGLE_SEARCH_ENGINE_ID=your_search_engine_id
```

### Установка зависимостей

```bash
pip install beautifulsoup4 lxml
```

## Примеры использования

### Поиск товаров со скидками
```bash
curl "http://localhost:8000/gifts/search?query=iphone&budget=50000&limit=5"
```

### Поиск с фильтрацией по скидкам
```bash
curl -X POST "http://localhost:8000/gifts/search" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "наушники",
    "budget": 10000,
    "filter": {
      "has_discount": true,
      "min_discount_percent": 20,
      "min_rating": 4.0
    }
  }'
```

## Ограничения

- Wildberries и Ozon не имеют официальных публичных API
- Google Shopping требует платного API ключа
- Веб-скрапинг может быть нестабильным при изменении структуры сайтов
- Рекомендуется использовать кэширование для улучшения производительности

## Точность цен

- **Wildberries**: 99% точность (официальное API)
- **Ozon**: 85-95% точность (зависит от структуры сайта)
- **Google Shopping**: 90% точность (зависит от источника)
- **Fallback**: ссылки на страницы поиска (цены не доступны)

Система обеспечивает **максимально точные цены** из доступных источников и автоматически отфильтровывает некорректные данные.
